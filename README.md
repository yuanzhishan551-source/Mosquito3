# Mosquito3

😤 **Stress Buster** - 一个使用手势识别的互动压力释放游戏

## 🎮 游戏介绍

通过摄像头和手部追踪技术，你可以用真实的手势来"打"掉屏幕上漂浮的压力词汇（如：考试、账单、截止日期等）。

### 游戏操作

- 🖐️ **拍打**：将手掌推向屏幕
- 🙏 **拍手**：用双手夹击目标

**演示链接**：[https://yuanzhishan551-source.github.io/Mosquito3/](https://yuanzhishan551-source.github.io/Mosquito3/)

## 🛠️ 技术栈

- MediaPipe Hands - 手部追踪
- HTML5 Canvas - 图形渲染
- JavaScript - 游戏逻辑

## 📋 使用说明

1. 访问演示链接
2. 点击"Click Here to Start"
3. 允许浏览器访问摄像头
4. 开始用手势释放压力吧！

## 🚀 本地运行

直接用浏览器打开 `index.html` 文件即可。
